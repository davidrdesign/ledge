# Ledge — Inventory

A single-file inventory management app built in HTML, CSS, and JavaScript. No dependencies, no build step — open `inventory-app.html` in a browser and it works.

## Features

- **Dashboard** — overview stats and activity feed
- **Inventory** — item list with detail panel, pricing, and stock levels
- **Purchase Orders** — track open and closed POs
- **Manufacturing** — work orders with BOM and progress tracking
- **Suppliers** — vendor list and contact info
- **Responsive** — works on desktop, tablet, and mobile

## Usage

Open `inventory-app.html` directly in any modern browser. No server required.
